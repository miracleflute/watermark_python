# watermark 사용법 및 예시


## 사용법
각 파일과 폴더의 경로는 한글이 없어야 합니다.


watermark로 사용할 이미지는 png파일을 권장합니다.



적용 및 저장 버튼을 누르면 미리보기가 나오고 미리보기를 끄면 저장이 됩니다.
